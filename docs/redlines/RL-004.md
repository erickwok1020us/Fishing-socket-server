# RL-004: No HP Alteration

TYPE: RED_LINE
STATUS: RESOLVED
SCOPE: GLOBAL
AI_ACTION_ALLOWED: 1

## STATEMENT

Client CANNOT alter fish HP values. HP is determined and tracked exclusively server-side.

## SOURCE

Bible M4: "Forbidden: No altering HP"

## VIOLATION RESPONSE

STOP. HP manipulation invalidates all kill outcomes.

## TEST CRITERIA

- Client-submitted HP values must be ignored
