# RL-003: No RNG Alteration

TYPE: RED_LINE
STATUS: RESOLVED
SCOPE: GLOBAL
AI_ACTION_ALLOWED: 1

## STATEMENT

Client CANNOT influence or alter the Random Number Generator. All RNG is server-side CSPRNG only.

## SOURCE

Bible M4: "Forbidden: No altering RNG"

## VIOLATION RESPONSE

STOP. Any client-influenced RNG invalidates all game outcomes.

## TEST CRITERIA

- Client-submitted seed values must be ignored for game outcomes
- Math.random() must never be used for payout decisions
