TYPE: RED_LINE
STATUS: RESOLVED

## STATEMENT

No hidden difficulty scaling. Game difficulty MUST NOT change based on player's balance, win history, or session time.

## SOURCE

Bible Gambling Psychology: "Forbidden: Hidden difficulty scaling"

## VIOLATION RESPONSE

STOP. Remove any adaptive difficulty that targets individual players.

- Fish HP, spawn rates, and RTP must be identical regardless of player balance
- No code path that adjusts game parameters based on player win/loss history
